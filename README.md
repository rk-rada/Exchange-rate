# Kursy-walut
Aplikacja pobierająca i przetwarzająca informacje z NBP o kursach walut.

# Zakres pracy:
- wykorzystanie metod numerycznych
- prosty interfejs użytkownika


# Oprogramowanie i technologie:
- Anaconda Prompt
- Spyder
- Python
